\# 👑 Crown Streamer



A password-protected, free anime streaming website built with Node.js, Express, and the Consumet API.



\## 🚀 Live Demo

\*\*URL\*\*: \[https://crown-streamer.onrender.com](https://crown-streamer.onrender.com)



\*\*Access Password\*\*: Set by the site administrator (environment variable `ACCESS\_PASSWORD`).

\## 🛠️ Tech Stack

\- \*\*Backend\*\*: Node.js, Express

\- \*\*Frontend\*\*: HTML, CSS, JavaScript

\- \*\*APIs\*\*: Consumet (Gogoanime provider)

\- \*\*Hosting\*\*: Render (Free Tier)



\---



\## 🔐 Security Features

\- Password-protected entry gate.

\- Session-based authentication.

\- IP rate-limiting to prevent scraping.

\- Frontend code obfuscation (via Webpack/Terser).
